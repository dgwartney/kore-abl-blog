## ABL Developer Blog

This blog covers **Agent Blueprint Language (ABL)** — the declarative language for building intelligent agents on the [Kore AI Agent Platform](https://kore.ai).

Topics include:

- ABL syntax deep-dives and language patterns
- Tool and MCP server integration
- Multi-agent orchestration (supervisor, adaptive network)
- Memory and state management strategies
- Safety guardrails and output validation
- Testing and debugging agents
- Real-world agent design case studies

## About the Author

**Dave Gwartney** is a Solutions Engineer at Kore AI, specializing in agentic application design and the ABL platform. He maintains the [ABL Developer Guide](https://github.com/dgwartney/kore-abl-developer-guide) and writes here about patterns and practices learned in the field.

Questions or feedback? Open an issue on [GitHub](https://github.com/dgwartney/kore-abl-blog).
